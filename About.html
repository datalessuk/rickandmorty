About


This is a Rick and Morty Character search written by Mark Harrison. I have used the Rick and Morty API This gives you access to hundreds of characters from the show with some useful information about them.

Why I built this


Because i am a die hard Rick and Morty fan and aspiring front end developer so why not ?


Technologies used

Vanilla javascript has been used with HTML and bootstrap CSS for the front end.The Rick and Morty API has been used to fetch all of the shows characters.
